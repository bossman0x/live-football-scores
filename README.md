# ⚽ Live Football Scores

A simple web app that fetches and displays Premier League match results using [TheSportsDB API](https://www.thesportsdb.com/).

## Features
- Fetches real match data (teams, scores, date, time, venue)
- Displays team badges for each match
- Clean, dark-themed responsive UI
- Handles loading and error states gracefully

## Built With
- HTML
- CSS
- JavaScript (Fetch API)
- [TheSportsDB API](https://www.thesportsdb.com/)
